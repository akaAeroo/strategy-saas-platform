# strategy-saas-platform
---

## 公司大模型配置

本项目支持使用公司内部的 Minimax API Key。

### 快速配置

```bash
# macOS / Linux
./bin/strategy-config.sh

# Windows PowerShell
.\bin\strategy-config.ps1
```

详细配置说明请参考 [CONFIG_GUIDE.md](CONFIG_GUIDE.md)

### 获取 API Key

1. 访问公司网关：https://gateway.xrtan.cn:5029
2. 使用钉钉扫码登录
3. 进入「可用模型」页面复制 API Key
